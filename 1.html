The Comprehensive Guide To AppArmor
AppArmor is an mandatory access control (MAC) like security system for Linux. It is designed to work with standard Unix discretionary access control (DAC) permissions while being easy to use and deploy, by allowing an admin to confine only specific applications.

Building a secure Ubuntu system: Four parts tutorials
This is an excellent serial of tutorials on how to build a secure Ubuntu as your primary OS, which involves all sorts of defence techniques.

OAuth 2.0​
OAuth 2.0 is the industry-standard protocol for authorization. OAuth 2.0 focuses on client developer simplicity while providing specific authorization flows for web applications, desktop applications, mobile phones, and living room devices. 

Relationship-based access control
Attached Files:
File 10.1.1.387.5553.pdf 
Relationship-based access control has received great attention over the last few years and it is an access control model underlying the popular social networking sites such as Facebook, Twitter, etc. The attached paper presents the model in a formal way. 

Inference control
Attached Files:
File SEv3-ch11-7sep.pdf 
The concept of inference control has been extensively studied in the context of statistical database. With the emergence of social networking sites, and information shared and published in these sites, how can we control your secret information being inferred by others. The book chapter attached gives a great introduction on this topic.

The optional texts for this module are:
